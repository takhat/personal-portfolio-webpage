My Latest Projects

Ada Capstone Project: Maze Game

A maze game built using Python along with Pygame library and deployed on Trinket. As the user enters the game, a maze is generated dynamically and randomly. The user can interact with the maze by using the player sprite and arrow keys to navigate the maze. There is also a “View Solution” feature. The game has 3 levels of complexity.

Inspiration Board

A full stack app representing an Inspiration (White)Board that displays a list of boards to select from, and returns all the “sticky notes/cards” with inspirational messages associated with the selected board. It has CRUD routes for the board and CRD routes for the card. It also has a “like” feature for the cards. The app is built using React frontend and uses Flask and Postgres db for the backend.
